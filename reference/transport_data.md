# Transport data

Simple example dataset containing transport data, used to demonstrate
various table styles in the example app.

## Usage

``` r
transport_data
```

## Format

A data frame with 10 rows and 5 variables:

- months:

  The month the data belongs to

- colours:

  The colour of the vehicle

- bikes:

  Cost of bikes

- vans:

  Cost of vans

- buses:

  Cost of buses

## Source

Random data generated for the shinyGovstyle package
