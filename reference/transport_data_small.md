# Transport data small

Smaller example dataset containing transport data, used to demonstrate
various table styles in the example app.

## Usage

``` r
transport_data_small
```

## Format

A data frame with 3 rows and 3 variables:

- months:

  The month the data belongs to

- bikes:

  Cost of bikes

- cars:

  Cost of cars

## Source

Random data generated for the shinyGovstyle package
